# Mozilla Apps Enhancements
 
## Firefox   
 
* NVDA+A (desktop) o NVDA+Control+A (Laptop) legge l'indirizzo della pagina. Se premuto due volte rapidamente, lo copia negli appunti. 
* NVDA+Fine (desktop) o NVDA+Shift+Fine (laptop) legge la barra di stato. Se premuto due volte rapidamente, copia il contenuto negli appunti. 
* NVDA+F8 Mostra un elenco di schede aperte. Se premuto due volte rapidamente, mostra i pulsanti della barra degli strumenti. 
* é possibile anche assegnare, dalle preferenze di NVDA, un gesto alternativo più comodo della doppia pressione per mostrare i pulsanti della barra degli strumenti. 
* NVDA+Control+N Legge l'ultima notifica e, se possibile, vi porta il focus di sistema. Premere due volte rapidamente per mostrare la cronologia delle notifiche. 
* NVDA+F6 Sposta il focus sul documento. 
 
## Thunderbird  
 
* In una finestra del messaggio:
	* Control+Shift+(1-9) Legge il mittente ed i destinatari del messaggio. Se si preme due volte rapidamente, apre il relativo menu Opzioni. 
	* Alt + Control + Shift + 1-9 Apre ugualmente il relativo menu delle opzioni; va usato se la doppia pressione non funziona correttamente.
	* Control+Shift+A Se presenti, porta il focus sull'elenco degli allegati.
	* NVDA+F6 Porta il focus sul corpo del messaggio.
 (Gli script seguenti sono disponibili nell'elenco dei messaggi se si attiva il riquadro di anteprima.) 
* Nell'elenco dei messaggi:
	* Se il riquadro di anteprima è attivato, premi NVDA + freccia giù (desktop) o NVDA + A (laptop) per leggere il messaggio senza lasciare l'elenco.
	* Control+NVDA+(1-9) Legge le informazioni delle colonne corrispondenti.
 
I tasti rapidi possono essere personalizzati in NVDA> preferenze> gesti e tasti di immissione quando Firefox o Thunderbird sono aperti.
In preferenze NVDA> Impostazioni> Mozilla Thunderbird è possibile abilitare/disabilitare la lettura automatica dei messaggi nel riquadro di anteprima.
